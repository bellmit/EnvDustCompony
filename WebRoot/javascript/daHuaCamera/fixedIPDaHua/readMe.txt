web_demo使用说明：
   1.双击webplugin.exe文件进行安装；
   2.浏览器打开web_demo文件夹中的index.html文件，即可看demo功能。
   3.web_demo中的接口使用说明参见文档《二次开发使用 WEB32网页调用接口说明V2.0.doc》及附件文档《网络SDK开发手册.chm》
   
关于几个功能说明：
1、控件初始化 需要添加 监听 AddEventListener，回调可以看文档的事件

2、监视和回放 相互切换 ，需要调用 SetModuleMode 接口

3、回放时候，必须要将录像查询回来才能回放

4、云台控制参数参见
参数：通道号、命令（见云台参数第四页 dwPTZCommand 列）、参数3(见云台参数第四页 param1列)、
参数4(见云台参数第四页 param2列)、参数5(见云台参数第四页 param3列)、参数6(见云台参数最后1页 是否停止，对云台八方向操作及镜头操作命令有效，进行其他操作时，本参数应填充FALSE)
